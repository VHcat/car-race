# Come on! Racing 🏎️

一个纯前端的赛车小游戏，使用 HTML5 Canvas 实现，无需任何后端服务，即开即用。

## 游戏特性

- **多种赛道**：乡村公路、高速公路、沙漠公路、城市街道、雪山公路、海边公路
- **8种车辆**：运动跑车、越野吉普、F1赛车、Monster Truck、警车、出租车、皮卡、装甲车
- **道具系统**：磁铁、保护盾、备用油箱、雷霆清除
- **商店系统**：购买道具和车辆
- **银行抽奖**：三档投注，赢取金币和道具
- **每日奖励**：14天签到奖励
- **任务系统**：完成各种成就获取奖励
- **数据持久化**：使用 localStorage 保存游戏进度

## 操作方式

- **键盘**：方向键 或 A/D 控制方向，P 或 Esc 暂停
- **触屏**：按住屏幕左右拖动控制车辆
- **数字键 1-4**：快速使用道具

## 在线体验

访问 GitHub Pages 即可在线游玩：[https://vhcat.github.io/car-race/](https://vhcat.github.io/car-race/)

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- 无需任何框架和构建工具
