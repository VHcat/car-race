# Come on! Racing 🏎️

一个纯前端的竖版赛车躲避小游戏。HTML5 Canvas 渲染，无需任何后端与构建工具，打开即玩。

在线体验：<https://vhcat.github.io/car-race/>

## 玩法简介

俯视竖版卷轴赛车：操控屏幕下方的赛车躲避迎面车流，油量随时间消耗（耗尽即结束），沿途收集金币、拾取道具，跑得越远难度越高。每 500 米经过检查站补给油量与金币。

核心操作：

- 键盘：`A`/`D` 或 `←`/`→` 转向，`空格`/`W`/`↑` 氮气加速，`F`/`Shift` 飞行（需装备飞翼），`1`-`6` 使用道具，`P`/`Esc` 暂停
- 触屏：手指按住屏幕，赛车跟随手指横向位置移动（跟手式操控）；右下角氮气按钮长按加速，装备飞翼后右侧出现 FLY 按钮，长按可飞越来车
- 移动端针对手感做了动态插值跟手、防误触、防下拉刷新等优化

## 游戏系统

- 6 条赛道：乡村公路、沿海公路、高速公路、城市夜景、沙漠公路、雪山公路，各自拥有独立主题画面（昼夜、天气、视差远景）与难度/金币倍率
- 10 台车辆：运动跑车、越野吉普、出租车、F1、皮卡、Monster Truck、警车、肌肉车、黄金跑车、装甲车，可在车库购买并升级 4 项属性（极速/加速/操控/氮气，各 5 级）
- 改装车间：7 种零件可购买/升级（各 5 级）并自由装配——飞翼装置（长按飞行，空中免疫碰撞、可捡金币，消耗能量与燃油）、装甲板、强化油箱、钕磁铁、金币芯片、涡轮机、氮气回收器；槽位随等级解锁（初始 2 个，LV.5/LV.10 各 +1）
- 氮气系统：险胜（擦身而过）积攒氮气与连击，连击越高得分倍率越高；氮气状态下可直接撞毁敌车
- 道具系统：磁铁、保护盾、备用油箱、雷霆清场、双倍金币、时间减缓（商店购买，局内 `1`-`6` 使用）
- 增益卡：起步满氮气、双倍金币卡（宝石购买，开局自动生效）
- 银行抽奖：三档投注老虎机，每日一次免费抽奖
- 每日任务：每天 3 个随机任务；14 天签到奖励；14 项永久成就
- 等级与头衔：里程换取经验，升级奖励金币与宝石
- 复活机制：撞车后可花费金币/宝石原地复活一次
- 数据持久化：localStorage 自动存档（v2 存档结构，兼容旧版数据迁移）

## 操作与设置

游戏内左上角暂停；设置页可开关音效/音乐/震动反馈、切换画质（高/低，低端设备建议低画质）、重置存档。

## 项目结构

```
car-race/
├── index.html        # 页面骨架（全部界面与弹层）
├── css/
│   └── style.css     # 视觉设计系统、动效、响应式布局
└── js/
    ├── data.js       # 配置数据：车辆/赛道/主题/道具/任务/成就 + 工具函数
    ├── save.js       # 存档读写、每日任务重置、统计上报
    ├── audio.js      # WebAudio 全合成音效/背景音乐/引擎声
    ├── car.js        # 赛车与敌车的 Canvas 绘制
    ├── ui.js         # 界面路由、菜单动画场景、车库/商店/银行/奖励/任务/设置
    ├── game.js       # 游戏核心循环、物理、渲染、新玩法系统
    └── main.js       # 全局输入绑定与初始化
```

脚本按依赖顺序以传统 `<script>` 标签加载（非 ES Module），因此直接双击 `index.html`（`file://` 协议）也能运行。

## 技术要点

- 纯 HTML5 + CSS3 + JavaScript，Canvas 2D 渲染，零依赖、零构建
- 帧率无关的物理更新（delta-time 归一化），高刷新率屏幕不会变快
- devicePixelRatio 适配（上限 2），画面清晰且兼顾性能
- 触屏跟手操控采用动态插值系数，随距离变化，操控细腻
- 音效全部由 WebAudio 实时合成（振荡器 + 噪声缓冲 + 包络），无任何音频资源文件
- 移动端：`100dvh` + `visualViewport` 适配、安全区适配、震动反馈、切后台自动暂停
